Common Shortcuts

    Double Click  = Select Word
    Triple Click = Select Line
    SHIFT + Click = List Select
    CTRL / CMD + Click = Multi Select
    CTRL / CMD + A = Select All
    CTRL / CMD + C = Copy
    CTRL / CMD + X = Cut
    CTRL / CMD + V = Paste
    CTRL / CMD + Z = Undo
    CTRL / CMD + Y = Redo
    CTRL / CMD + S = Save
  
    
Chrome Shortcuts more

    F5 = Refresh
    Scroll + Click = Close Tab more
    CTRL + U = View Source
    CTRL + SHIFT + I = Open Chrome Dev Tools
    CTRL / CMD + SHIFT + C = Inspect Element more
    CTRL / CMD + SHIFT + M (On Dev Tool Window) = Responsive View
    CTRL / CMD + SHIFT + T = Restore Chrome Tabs

    
Windows Shortcuts

VS Code Shortcuts

    SHIFT + Double Click File = Open in New Tab
    ALT / OPTIONS + Click File = Open Tab in Another Column
    CTRL / CMD + B = Toggle Sidebar Visibility
    ALT / OPTIONS + Z = Toggle Wordwrap
    SHIFT + ALT / OPTIONS + A = Insert a Comment
    CTRL / CMD + / = Comment Line or Selection
    CTRL / CMD + SPACE = Trigger Suggest
    ALT / OPTIONS + D = Duplicate Selection
    SHIFT + ALT / OPTIONS + ↑/↓ = Duplicate Line ↑/↓
    ALT / OPTIONS + E = Move Cursor to Line End
    ALT / OPTIONS + SHIFT + E = Select up to Line End
    ALT / OPTIONS + S = Move Cursor to Line Start
    ALT / OPTIONS + SHIFT + S = Select up to Line Start
    CTRL / CMD + ENTER = Insert After Line
    CTRL / CMD + SHIFT + ENTER = Insert Before Line
    ALT / OPTIONS + ↑/↓ = Move Line ↑/↓
    CTRL / CMD + Click Line = Multiline Edit
    SHIFT + ALT / OPTIONS + Click and Drag = Multiline Edit Selection ↑/↓
    CTRL / CMD + ] = Indent Line
    CTRL / CMD + [ = Unindent Line
    CTRL / CMD + L = Selects the Entire Line
    CTRL / CMD + SHIFT + K = Delete Entire Line
    CTRL / CMD + ALT / OPTIONS + ↑/↓ = Increment / Decrement by 0.1
    CTRL / CMD + ↑/↓ = Increment / Decrement by 1
    CTRL / CMD + SHIFT + ↑/↓ = Increment / Decrement by 10
    CTRL / CMD + D = Find and Highlight The Next Selected Word
    CTRL / CMD + SPACE = Trigger Suggest
    CTRL / CMD + SHIFT + A = Select Inside Tags
    CTRL / CMD + SHIFT + SPACE = Expand Selection
    CTRL / CMD + SHIFT + P = Show All Commands | Useful for calling Emmet: Expand Abbreviation
    CTRL / CMD + K, CTRL / CMD + 8 = Fold Marker Regions
    CTRL / CMD + K, CTRL / CMD + 9 = Unfold Marker Regions
    CTRL / CMD + K, CTRL / CMD + I = Show Hover
